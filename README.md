# TvShowsApp

Project that shows the latest and most populars Tv Shows with The Movie Database API.  

Project from [Escalab Academy](https://escalab.tech/) - Full Stack MERN

## Tech

* React   v18.2.0
* Node    v16.14.2
* Yarn    v1.22.19
* Vite    v4.0.0
